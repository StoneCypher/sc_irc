channel_type/1

channel_type(ChannelName::[char()]) -> local | global

Describes a channel by type (local or global,) or throws if not a valid channel name. To be valid, a channel must start with # or &, must not contain a space, a control-g, a comma, and must be 200 or fewer characters including the sigil. The standard is vague on whether just-the-sigil is legitimate, so this library tolerates it.
